# The Story of CyberTHUG · 사이버더그 이야기

> 기억을 지운 채, 어느 가상세계로 "**전송**"된 떠그(THUG)들의 이야기.
> 맨손으로 땅을 파고, 수수료를 뜯기고, 아침 6시면 리셋되는 세계 —
> 코인판을 쏙 빼닮은 그곳의 연대기.

## 📖 어떤 이야기인가

약해 보이면 죽는 곳. 착한 짓도, 착한 척도 금지된 곳.
기억을 잃은 화자 **세르게이**가 이 흉흉한 가상세계 *CyberTHUG* 에 던져져,
야마도라·카이저소제 같은 이상한 동료들과 하루하루 땅을 파며 살아남는 이야기다.

블록체인과 코인판의 풍경 — 전송, 블럭, 컨펌, 채굴, 수수료, 리셋 —
을 다크 코미디로 비틀어 담은 연재 웹소설.

## 🧩 이 세계의 말들

| 용어 | 뜻 |
|---|---|
| **떠그 (THUG)** | 기억을 지운 채 이 세계로 전송된 주민 |
| **전송 · 블럭 · 컨펌** | 이 세계에 입장하는 절차. 컨펌 20회면 전송 완료 |
| **채굴** | 맨손·삽으로 땅을 파 흙을 모아 돈으로 바꾸는, 이곳의 유일한 노동 |
| **수수료** | 하키맨이 채굴한 흙의 절반을 떼어가는 것 |
| **리셋** | 죽어도 아침 6시면 되살아난다. 단, 빼앗긴 것과 빚은 그대로 남는다 |
| **브로** | 이곳엔 나이도 지위도 없다. 모두 형제(brother) |
